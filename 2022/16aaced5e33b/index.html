【闲话】 PB 没换架构

本文最后更新于：a day ago

背景

本来吧，Fagri 发现有一个不好的家伙入侵了 PB[1] 的服务器（再次给良心云[2]的告警服务点赞👍），懒如 Fagri 直接在良心云上重装了一下系统。

非常巧的，Fagri 的个人 PC 也重装了一下系统，原来 Hexo 的文件全无（虽然我是备份了_config.yml; _config.fluid.yml; sources Directory）。

所以 Remote 端和 Local 端的 Hexo 都需要重新配置了。
这是前言。

事件始末

后来，Fagri 发现了一个看起来非常棒的东西 —— Gridea，遂弃 Hexo 而改用 Gridea.

但是，既然为了这个事情，Fagri 专门写了一篇 article 出来，就说明不可能这么简单。

是的，在配置好 Gridea 的所有 Config 后，虽然客户端显示同步成功，但 GitHub 上 什 么 反 应 都没有。故弃坑回转 Hexo.

然后就是你们看到的这个站点啦！

Fagri 把它叫做 PB2.0船新版本PB 2.0 船新版本PB2.0船新版本.
