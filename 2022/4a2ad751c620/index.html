C# 从入门到入土（一）

本文最后更新于：a month ago

0x00 背景

如你所见，这是一篇 C# 半小白教程.

我要求你在阅读这篇教程之前：

  • 知道并领会「提问的智慧」.

我希望你在阅读这篇教程之前：

  • 有学过 C++ 或类似的具有相同原理（OOP[1]）的编程语言.
  • 知道 C Sharp[2] 在 CSP-J/S 以及 NOIP 均不能用.[3]

Ready? Let’s Go!

0x01 环境配置

众 所 周 知，要想写出好的代码首先要有一个好的环境，于是乎让我们开始配置环境吧~

以下内容部分参考 菜鸟教程.

TL;DR. (CSharp Compiler)

下面为 TL;DR.[4] 内容.
如果你在用 Windows 操作系统，那么直接 Win + R 调出 cmd 窗口.

输入 csc 你要编译的文件的文件名，比如说 csc hellopb.cs 即可.这样就会生成一个hellopb.exe程序.

如果得到 'csc' is not recognized as an internal or external command, operable program or batch file. 之类的提示，可以参考菜鸟教程下方的这篇笔记. 如果还是不行，可以下方评论.

Visual Studio

与上面的 TL;DR 方法可以并存，但二选一即可.

请参考 Microsoft 技术文档. 它比我写的好多了

但请注意，在进行 第 4 步 - 选择工作负载 时，务必选中 「.NET 桌面开发」选项. 其余自由配置.
